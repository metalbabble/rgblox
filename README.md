# RGBlox

RGBlox is a game based on classic falling-block puzzle games like Tetris, Dr.Mario, and Puyo-Puyo.

## Requirements and Specifications

All of the specific requirements, specifications, behaviors, and gameplay specifics are contained in various markdown (md) files in the [specifications](/specifications/) directory.

The markdownfile [general.md](specifications/general.md) provides a high-level oveview.
